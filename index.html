Проєкт: Програмно-апаратний інтерфейс для візуалізації стану довкілля на базі ESP32

Цей проєкт — це інноваційна IoT-система для моніторингу та візуалізації параметрів довкілля, розроблена з використанням мікроконтролера ESP32 та набору датчиків. Основна мета — збирати дані про температуру, вологість, якість повітря (включаючи рівень CO2), освітлення та шум у реальному часі, з подальшою передачею їх на хмарний сервер для аналізу та відображення.

Ключові компоненти та функції:

  • Апаратна частина: ESP32 як центральний процесор, інтегрований з датчиками (DHT22 для температури/вологості, MQ-135 та CCS811 для газів, BH1750 для освітлення, мікрофон для шуму) та OLED-дисплеєм для локальної візуалізації.
  • Програмна частина: Дані передаються через Wi-Fi на MQTT-брокер (HiveMQ Cloud), де вони обробляються та доступні для веб-сайту на Render.com з графіками та сповіщеннями.
  • Застосування: Ідеально для шкільних класів — моніторинг допомагає підтримувати комфортні умови, виявляти забруднення та оптимізувати енергоспоживання.

Значення проєкту:

У контексті глобальних викликів, таких як зміна клімату та забруднення, проєкт сприяє досягненню Цілей сталого розвитку ООН (SDG 3: здоров'я, SDG 11: сталі міста, SDG 13: боротьба зі зміною клімату). Він підвищує обізнаність про екологічні проблеми, допомагає запобігати респіраторним захворюванням від поганої вентиляції та заохочує до енергоефективності. Для освіти — це практичний інструмент для вивчення IoT, програмування та екології, з потенціалом для масштабування в міських мережах моніторингу.

Проєкт демонструє, як доступні технології можуть перетворювати дані на дії для кращого майбутнього!
